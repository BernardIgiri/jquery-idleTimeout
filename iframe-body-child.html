Iframe Inserted into Document Body Page

This is a demonstration of an iframe dynamically inserted into the body of the document.

The page within this iframe is from the same domain as the parent page.

If an 'iframe recheck' is called, activity events within this iframe can be 'detected'.

Iframes inserted into the body of the document require special handling. Add this snippet to the function which inserts the iframe: $.fn.idleTimeout().iframeRecheck();.
